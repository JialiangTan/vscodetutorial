Tutorial 4: Remote Development using SSH

May 20, 2022

Visual Studio Code allows you to open a remote folder on any remote machine, virtual machine with a running SSH server.

As a NC State student, you have access to log into XXX!!! cluster and code on it. Once connected to a sever, you can interact with files and folders anywhere on the remote filesystem.

Also, XXX!!! cluster already has Python installed, there is no need to install Python yourselves on remote machine.

Step 1: Install the Remote - SSH extension

Let’s open the VS Code.

Click the Extension button on the left menu to open the Visual Studio Marketplace.

In the Marketplace, search remote.

Choose the Remote - SSH extension developed by Micosoft.

Install it, and you will find the extension is installed and enabled.
